# نظام تسجيل الطلاب

تطبيق ويب لإدارة تسجيل الطلاب باستخدام React و Node.js و MongoDB Atlas.

## المميزات

- إضافة طلاب جدد
- عرض قائمة الطلاب
- تعديل بيانات الطلاب
- حذف الطلاب

## متطلبات التشغيل

- Node.js
- MongoDB Atlas حساب
- npm
